For loop sama saja dengan While loop namun beda pada strukturnya saja Contoh struktur dari for Loop: - statement1: berisi variabel atau nilai awal - statement2: berisi kondisi dari perulangan - statement3: berisi increment/decrement; for(statement1 ; statement2 ; statement3){ aksi; } For Of: sama seperti for loop namun lebih simpel; contoh struktur: for(var nameVar of name){ } contoh: let name = ["Maulana", "Malik", "Ibrahim"]; for(const hai of name){ console.log(hai); } "Maulana" "Malik" "Ibrahim"
